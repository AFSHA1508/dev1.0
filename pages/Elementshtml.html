HTML - ELEMENTS

An element is a part of a webpage. In XML and HTML, an element may contain a data item or a chunk of text or an image, or perhaps nothing. A typical element includes an opening tag with some attributes, enclosed text content, and a closing tag.

Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<em> emphasized <em>

Note : In HTML, a self-closing element is written simply as <br>. In XHTML, a self-closing element requires a space and a trailing slash, such as <br />.

Nesting HTML Elements

It is often necessary to code certain tags (and their text) within the definition of other tags (between the start and end tags).

Element inside another element is bascially known as nesting of elements.

All HTML documents contain four HTML elements ( <html> , <head> , <title> ,<body> )




Note : Never Skip the End Tag. Some HTML elements will display correctly, even if you forget the end tag.
