HTML Attributes

HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of an HTML element type.


So far we have seen few HTML tags and their usage like heading tags <h1>, <h2>, paragraph tag <p> , image tag <img>and other tags. We used them so far in their simplest form, but most of the HTML tags can also have attributes, which add extra effect to the information.

Attributes are made of two parts : a Name and it's Value.

  • Name - is the property you want to set .For instance, the paragraph <p> element in the example carries an attribute whose name is align, which you can use to indicate the alignment of paragraph on the page.
  • Value - is what you want the value of the property to be set and always put within quotations. The below example shows three possible values of align attribute: left, center and right.
